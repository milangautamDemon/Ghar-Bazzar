# Client

This is the frontend part of the project.

## What Is There

- React
- TypeScript
- Vite
- Login page
- Signup page
- Dashboard page

## How To Run

```bash
cd client
npm install
npm run dev
```

Client runs on:

```text
http://localhost:5173
```

## Important

- Backend should also be running
- Client uses API `http://localhost:8000/api`
