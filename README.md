# Junior Full Stack Developer Assignment

This project has:

- `client` for frontend
- `server` for backend

## What Is There

### Client

- React
- TypeScript
- Vite
- Login page
- Signup page
- Dashboard page

### Server

- Node.js
- Express
- TypeScript
- MongoDB
- Authentication APIs
- Favourite APIs

## How To Run

Open two terminals.

### Terminal 1

```bash
cd server
npm install
npm run dev
```

Server runs on:

```text
http://localhost:5000
```

### Terminal 2

```bash
cd client
npm install
npm run dev
```

Client runs on:

```text
http://localhost:5173
```

## Important

- MongoDB should be running
- Client uses backend on `http://localhost:5000`
